FluAntigen

FluAntigen aim to provide the user with useful tools for antigenic relation checking and browsing of antigenic data. Currently avaialbe tools are as below:


Database Searching & Retrieval

The collection of influenza antigenic data which are used in training of IAV-CNN machine learning model and conscturion of graph representation. Various optons and filter are given to customize the searching and results are presented in the table form with useful tools and buttons.



Search

Antigenic Relation Network

A knowledge graph representation of antigenic database powered by neovis.js and neo4j database. We introduce this new tools where you can browse and analyse our database and get better overview of how strains are related to each other in sequence similarity and antigenic distance. You will be able to make use of various filter to see different form of representations.

Antigenicity Prediction using IVA-CNN model

A 2D convolutional neural network (CNN) model to infer influenza antigenic variants (IVA-CNN). CNN architecture is constructed that enables networks to focus on informative residue features by fusing both spatial and channel-wise information with local receptive fields at each layer. Experimental results on three influenza datasets show IVA-CNN achieves state-of-the-art performance combing the new distributed representation with our proposed architecture.
